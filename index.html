Introducción

Esta segunda práctica de la asignatura consiste en la creación de una página web estática utilizando los conocimientos adquiridos en HTML y CSS. El objetivo es desarrollar un sitio web completo que, a través de una página principal y tres páginas interiores interconectadas, demostrando el manejo de diferentes elementos esenciales de HTML como títulos, párrafos, tablas, elementos multimedia y formularios web. Para el desarrollo de la misma, utilizaremos CSS de forma manual y con la ayuda de Bootstrap. Una vez finalizada, se publicará la página en GitHub. En resumen, este ejercicio combina la parte técnica y creativa del desarrollo web como iniciación a esta parte de la asignatura.
